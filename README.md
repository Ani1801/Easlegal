# EasLegal Partners - Legal & Financial Solutions

Empowering startups and founders with comprehensive legal, financial, and compliance solutions.

## Architecture

This project is structured as a mono-repo with distinct Client and Server applications.

### Directory Structure

- **/client**: Frontend application built with Vite, React, TypeScript, and Tailwind CSS.
- **/server**: Backend API built with Node.js, Express, and TypeScript.
- **/docs**: Project documentation, sitemaps, and assets.

## Getting Started

### Prerequisites

- Node.js (v18+)
- npm or pnpm

### Setup Client

```bash
cd client
npm install
npm run dev
```

### Setup Server

```bash
cd server
npm install
npm run dev
```

## Features

- **Responsive Design**: Professional UI/UX matching modern standards.
- **Global Navigation**: Unified Navbar and Footer across all pages.
- **Client-Side Routing**: Fast navigation with React Router.
- **Form Integration**: Ready for backend API integration.
